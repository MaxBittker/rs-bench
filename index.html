rs-bench

AI agent benchmark for RuneScape gameplay tasks

Code rs-sdk

rs-bench evaluates AI coding agents on their ability to play Old School RuneScape. Agents are given access to a game environment via rs-sdk and must write code to train skills, earn gold, and navigate the game world within fixed time horizons.

Skill XP — 30 min

Total XP gained across 16 skills in a 30-minute horizon. Click through for the full interactive view.

Benchmark Design

16 Skills

Each RuneScape skill is a separate task, measuring XP gained within 10-minute and 30-minute horizons.

Gold Earning

Agents must figure out how to earn coins, requiring multi-step planning and game knowledge.

Sandboxed Environment

Each run uses a fresh Docker container with the game server at 8x speed, ensuring reproducibility.

Multi-Model Comparison

Run the same tasks across multiple frontier models to compare agent capabilities.
